Quiz App

A simple and interactive Quiz Application built using HTML, CSS, and JavaScript.

Features

- Multiple-choice questions
- Instant answer selection
- Score calculation
- User-friendly interface
- Responsive design

Technologies Used

- HTML
- CSS
- JavaScript

Project Objective

This project was created to practice JavaScript concepts such as DOM manipulation, event handling, arrays, functions, and user interaction.

How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Run "index.html" in any web browser.

Future Improvements

- Timer for each question
- Randomized questions
- Difficulty levels
- High score tracking
- Local storage support
